# TORO 기획서

## 프로젝트 개요

친구들과 함께 사용하는 디스코드 서버에 자연스럽게 대화에 끼어드는 AI 봇.
단순한 명령어 기반이 아닌, 대화 맥락을 파악하고 적절한 타이밍에 반응하는 것을 목표로 한다.

---

## 향후 확장 아이디어

- **키워드 트리거** — 특정 단어 감지 시 특정 반응 (예: "배고파" → 치킨 추천)
- **게임 룰렛** — 오늘 뭐 할지 랜덤 뽑기
- **채널별 성격 분리** — 채널마다 다른 System Prompt 적용

---

## 개선 로드맵

상세 내용은 `TORO/plan/` 폴더 참고.

### AI 품질
- [[자동 압축]] — 오래된 히스토리를 AI 요약으로 교체, 맥락 유지력 향상
- [[구조화된 메모리]] — 유저/피드백/프로젝트/참조 타입 분류
- [[대화 히스토리 영속화]] — 봇 재시작 후에도 대화 맥락 유지

### 운영
- [[토큰 비용 추적]] — API 사용량 모니터링, 예산 초과 경고
- **환경변수 백업** — 프로젝트별 `.env` 파일을 Vaultwarden(NAS)에 Secure Note로 보관

### 기능
- [[음악 재생 기능]] — ✅ 구현 완료. yt-dlp + FFmpeg로 음성 채널 재생, autoplay, 검색 선택 UI, 장르/아티스트 추천
  - 향후: 유튜브 관련 동영상 API로 추천 고도화
- [[플레이리스트 기능]] — 좋아요(❤️), 커스텀 플레이리스트, 서버 공유 플레이리스트, 재생 기록 기반 생성
